# Airbnb Clone 🏠

A full-stack Airbnb Clone web application built using the MERN Stack (JavaScript, 
MongoDB, Express.js, React.js, Node.js).
This project allows users to explore property listings, add their own stays, and experience a modern booking platform UI inspired by Airbnb.

---

## 🚀 Features

* User Authentication & Authorization
* Add, Edit, and Delete Listings
* Upload Property Images
* Responsive User Interface
* Property Details Page
* Search & Explore Listings
* MongoDB Database Integration
* REST API using Express.js
* Dynamic Frontend
* Secure Backend with Node.js

---

## 🛠️ Tech Stack

### Frontend

* CSS3

### Backend

* Mongoose

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/Ishwarishelke1/Airbnb_clone_MERN_Project.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd Airbnb_clone_MERN_Project
```

### 3️⃣ Install Dependencies

#### Backend

```bash
cd backend
npm install
```
---

## ▶️ Run the Project

### Start Backend Server

```bash
npm start
```
